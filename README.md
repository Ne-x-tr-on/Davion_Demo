## 🏗️ How It Works

### 1. Create a Project
Add title, description, tech stack, and collaborators.

### 2. Log Your Progress
Track daily/weekly updates, upload files, and record milestones.

### 3. Collaborate
Invite teammates, assign roles, and build together.

### 4. Share & Get Hired
Generate a portfolio recruiters can verify.
